# Cinema Backend System

## Setup Instructions

### Prerequisites
- Node.js 18+
- Docker
- PostgreSQL client (optional)

### Installation
1. Clone the repository
2. Install dependencies:
```bash
npm install

Create .env file:

```
DB_HOST=localhost
DB_PORT=5432
DB_USER=cinemax_user
DB_PASSWORD=cinemax_password
DB_NAME=cinemax
PORT=3001
JWT_SECRET=your_jwt_secret
```
